# Smart IoT-Based Plasmonic PCF Biosensor

## Project Overview
This repository contains the web interface and documentation for a graduation project focused on designing a highly sensitive Photonic Crystal Fiber (PCF) biosensor integrated with an IoT network. The project aims to provide real-time, highly accurate biological sensing capabilities for medical diagnostics and health monitoring.

## Key Features
- Advanced Simulation: Numerical analysis and modal simulations performed using COMSOL Multiphysics.
- High Sensitivity: Optimized air-holes configuration and plasmonic layer for maximum Surface Plasmon Resonance (SPR) effect.
- IoT Integration: Cloud-connected architecture to enable real-time data transmission and remote monitoring.
- Interactive Dashboard: A responsive web interface developed to visualize optical modes, sensor data, and project documentation.

## Technologies & Tools
- Simulation & Modeling: COMSOL Multiphysics
- Web Interface: HTML5, CSS3, JavaScript
- Network/IoT: Cloud connectivity, routing infrastructure, and real-time data flow
