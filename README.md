# Smart IoT-Based Plasmonic PCF Biosensor 🔬🌐

## 📌 Project Overview
This project presents the design, simulation, and implementation of a highly sensitive Photonic Crystal Fiber (PCF) biosensor integrated with an Internet of Things (IoT) network. Developed as a graduation project, it aims to provide real-time, highly accurate biological sensing capabilities suitable for advanced medical diagnostics and continuous health monitoring.

## 🚀 Key Features
*   **Advanced Simulation:** Rigorous numerical analysis and modal simulations performed using **COMSOL Multiphysics**.
*   **High Sensitivity:** Optimized air-holes configuration and plasmonic layer for maximum Surface Plasmon Resonance (SPR) effect.
*   **IoT Integration:** Cloud-connected architecture enabling real-time data transmission and remote monitoring.
*   **Interactive Dashboard:** A responsive, user-friendly web interface developed to visualize optical modes, sensor data, and project documentation seamlessly.

## 🛠️ Technologies & Tools
*   **Simulation & Modeling:** COMSOL Multiphysics
*   **Web Interface:** HTML5, CSS3, JavaScript
*   **Network/IoT:** Cloud connectivity, Routing infrastructure, & Real-time data flow
